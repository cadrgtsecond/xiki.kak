= Xiki.kak
eXecutable Wiki

Xiki is a cross between a shell, a filetree and a plumber(a la plan9)
all inside of a regular text buffer.

Xiki.kak is a full implementation of https://www.youtube.com/watch?v=YufgfbUgEgI[Xiki] using Kakoune,
that does not depend on heavyweight dependencies like Ruby or Emacs.

See link:./doc/xiki.asciidoc[`:doc xiki`] for more information

== Installation
Xiki is written in Zig which is a required dependency for building.

=== With kak-bundle
[source,kak]
----
bundle xiki.kak https://github.com/cadrgtsecond/xiki.kak %{
    hook global WinSetOption filetype=xiki %{
        map window normal <ret> ': xiki<ret>'
        map window normal <a-ret> ': xiki-execute -fifo<ret>'
    }

    # Scratch buffer should use Xiki!
    hook global WinCreate '\*scratch\*' %{
        set-option window filetype xiki
    }
}
bundle-install-hook xiki.kak %{
    ./install
    cp -rf doc/ ~/.config/kak/autoload/xiki-doc
}
----

=== Manual installation
[source,bash]
----
cp -rf rc doc -t ~/.config/kak/autoload/
cp xiki ~/.local/bin
mkdir -p ~/.config/xiki
cp -rf default scripts -t ~/.config/xiki/
----
